API description
example.com/api?format=json (just prints the user’s IP)
example.com/api?format=json&geo=true (print detailed geo data of the user’s IP)
Or, to get the details of another IP:
example.com/api?ip=1.2.3.4&format=json
example.com/api?ip=1.2.3.4&format=json&geo=true
